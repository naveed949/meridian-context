# Phase 1 — tiny corpus

Short Markdown files for experimenting with **chunk → embed → retrieve → generate**.

Intentionally includes:

- **Near-duplicates** — overlapping wording across pairs of files.
- **Synonym / vocabulary variation** — same idea, different terms (`automobile` vs `car`).
- **Mixed topics** — retrieval fundamentals, RAG tradeoffs, and unrelated “noise” (recipes).

Paths are listed from `notebooks/phase1_dense_rag_baseline.ipynb` (Step 0).
